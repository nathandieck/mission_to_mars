# "Mission to Mars"

This repository allows the user to run a Flask app that populates a webpage with up-to-date information from a variety of sources gleaned using webscraping tools (Beautiful Soup and Splinter). Data sources include NASA and the Jet Propulsion Laboratory, the USGS, and Twitter. The user is able to click on a button embedded in the webpage to reload all contant at will. 

### Resources Employed
* HTML & CSS with Bootstrap
* Python
* Flask
* Beautiful Soup
* Splinter
